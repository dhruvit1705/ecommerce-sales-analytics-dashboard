# Interactive Sales & Customer Insights Dashboard

## 📊 Project Overview
This project presents an interactive Excel dashboard that analyzes global e-commerce sales data to uncover insights about customer behavior, sales performance, and product category trends.

The dashboard allows users to dynamically filter data using slicers and visualize key metrics such as orders by country, revenue by category, monthly sales trends, and customer segmentation.

---

## 🎯 Objectives
- Analyze e-commerce sales performance
- Identify top-performing product categories
- Understand customer purchasing behavior
- Visualize sales trends across different countries
- Build an interactive dashboard using Excel

---

## 📁 Dataset
The dataset contains **5000+ e-commerce transactions** including:

- Order ID
- Order Date
- Country
- Product Category
- Product Name
- Quantity
- Unit Price
- Customer Type
- Order Mode

Additional calculated fields were created:
- Revenue
- Month
- Year

---

## ⚙️ Tools & Techniques Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers for interactive filtering
- Data Cleaning
- Data Visualization

---

## 📈 Dashboard Features

The dashboard includes the following visualizations:

1. **Orders by Country**
2. **Revenue by Category**
3. **Monthly Sales Trend**
4. **Customer Type Share**

Interactive filters include:
- Country
- Category
- Customer Type
- Order Mode

Key Performance Indicators (KPIs):
- Total Orders
- Average Revenue

## 🚀 Key Insights

- Clothing and Furniture generate the highest revenue.
- The majority of orders come from countries such as the UK, Germany, and India.
- Returning customers contribute significantly to total sales.
- Sales trends vary significantly across months.
